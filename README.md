-Advanced-CLI-Contact-Management-System-with-Search-Filters
Advanced CLI Contact Management System is a Python-based mini CRM that allows users to add, view, search, filter, update, and delete contacts efficiently. It supports data validation, unique IDs, and file-based persistence using JSON/CSV. The system features a menu-driven interface, modular functions, and error handling for smooth user experience.
📇 Advanced CLI Contact Management System

 🎯 Objective
A professional Python-based Command Line Contact Management System that allows users to efficiently store, manage, search, and filter contacts. This project simulates a mini CRM system.
---

🚀 Features
🔹 Core Features

* ➕ Add Contact (with validation)
* 📋 View Contacts (formatted table)
* 🔍 Search Contacts (Name, Phone, Email)
* 🎯 Filter Contacts (City, Company)
* ✏️ Update Contact
* ❌ Delete Contact
* 💾 Save & Load Data (JSON)

🔹 Advanced Features

* ✅ Unique ID for each contact
* ✅ Case-insensitive search
* ✅ Menu-driven interface
* ✅ Exception handling
* ✅ Modular code structure

🌟 Bonus Features

* ⭐ Favorite contacts
* 📤 Export to CSV
* 🔄 Import from file
* 🔃 Sorting options

---

🛠 Technologies Used

* Python 3
* JSON (Data Storage)
* CLI (Command Line Interface)

---

▶️ How to Run

1. Install Python

2. Clone the repository:

   ```bash
   git clone https://github.com/your-username/cli-contact-management-system.git
   ```

3. Navigate to project folder:

   ```bash
   cd cli-contact-management-system
   ```

4. Run the program:

   ```bash
   python main.py
   ```
